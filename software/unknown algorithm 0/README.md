## Problem
- Given 2 ints `n` and `k`, return a list of `k` ints, such that their values are as close as possible to each other, and their product is `n`.
- If `n` is prime, return an empty array.

How to do it efficiently?

## Example
n = 1114112
k = 2
out = (1024 1088)
